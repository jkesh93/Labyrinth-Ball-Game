# Labyrinth-Ball-Game

### About
###### Labyrinth game for my COS377 (Android Development) class at UMF. Fall 2016.

### Features
* Debugging Mode 
* Background 
* Persistant High score (IP) 
* Menu (IP) 
* Ad Space (IP)
* Sounds (IP)
* Powerups (IP)

IP means In Progress.
